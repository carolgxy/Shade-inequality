# Global Shade Inequality Analysis

This repository contains code for analysing global shade distribution inequality and exploring its relationships with socioeconomic indicators. The workflow includes:

- Calculating the Gini coefficient for shade inequality.
- Building XGBoost regression models to estimate influencing factors.
- Using SHAP for model interpretation and visual explanation.
- Visualising results with Matplotlib and Seaborn.

## Dependencies

Install all required packages before running this code:
pip install pandas numpy matplotlib seaborn scikit-learn shap pygam scipy xgboost==3.0.5
